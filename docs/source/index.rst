Installation
==================
pip can be used for the installation:

`pip install isocompy`

Alternatively, download the isocompy folder from the github repository and add it to lib folder in python path alongside other python libraries.

Jupyter Notebook
==================
There is a notebook in [isocompy's github repository](https://github.com/IDAEA-EVS/Isocompy) that explains an example of implementing isocompy on spatial and isotopic data
